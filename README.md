# 📄 SmartDoc AI
*Intelligent Document Converter with Fuzzy Mamdani Logic*

SmartDoc AI adalah sebuah aplikasi web pintar berbasis kecerdasan buatan yang dirancang untuk mengonversi format dokumen (PDF, Word, Excel, PowerPoint) tanpa merusak tata letak (layout). Proyek ini dikembangkan khusus sebagai implementasi nyata algoritma AI untuk keperluan akademik (Mata Kuliah AI).

## ✨ Fitur Utama
1. **Multi-Format Conversion**: Konversi antar dokumen `.pdf`, `.docx`, `.xlsx`, dan `.pptx`.
2. **Fuzzy Mamdani Decision Engine**: Menggunakan algoritma logika *Fuzzy Mamdani* murni (Matematika) untuk menghitung kompleksitas dokumen berdasarkan metrik ukuran file dan kepadatan elemen (Tabel/Gambar/Paragraf).
3. **AI Layout Recovery**: Algoritma cerdas untuk memulihkan margin, font, dan grid yang sering hancur pada konversi konvensional.
4. **Interactive UI**: Antarmuka modern yang interaktif, *responsive*, dengan efek mikro-animasi (*hover-lift*, *drag-and-drop* file).
